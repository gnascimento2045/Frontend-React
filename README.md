# Projeto em Desenvolvimento

> v0.1.0 - Início do projeto

## Sobre

Projeto iniciando com sistema de autenticação... 

**Frontend**
- React 18
- Less

**Backend**
- AdonisJS 6
- PostgreSQL
- TypeScript

## Implementado

- Tela de Login 
- Tela de Cadastro
- Validações de formulário
- API RESTful para autenticação
- Hash de senhas
- Validação de email duplicado

## Instalação

**Backend**
```bash
cd backend
npm install
cp .env.example .env
node ace migration:run
```

**Frontend**
```bash
cd frontend
npm install
npm run dev
```
